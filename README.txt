
# Cara Deploy di Railway

1. Buat akun Railway di https://railway.app
2. Buat project baru, pilih 'Deploy from GitHub Repo'.
3. Upload semua file project ini ke GitHub repository kamu.
4. Tambahkan Environment Variable:
   - TELEGRAM_TOKEN = masukkan token bot kamu
5. Setelah deploy, buka Railway Console, jalankan:
   railway run playwright install
